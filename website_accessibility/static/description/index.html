Website Accessibility

Add missing accessibility features to your website.

Overview

The Accessibility module in odoo helps to implements the accessibility features for any Odoo Theme or Page Builder. While most accessibility issues can't be addressed without directly changing your content, this module adds a number of helpful accessibility features.

Features

Resize font (increase/decrease)

Links Underline

Grayscale Mode

Negative Contrast

Readable Font: Resets embedded fonts

High Contrast

Light Background

SCREENSHOTS

Click on accessibility icon to access the features.

Increase the font size across the website.

Decrease the font size across the website.

Apply grayscale effect across the website.

Apply high contrast effect across the website.

Apply light background effect across the website.

Apply links underline effect across the website.

Apply negative contrast effect across the website.

Resets embedded fonts across the website.

Resets all the applied effects.
